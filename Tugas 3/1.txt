10 contoh built-in function di javascript

1. toString() : Untuk mengubah type data menjadi string 
    contoh  toString(10)
            toString(true)

2. concat() : Untuk menggabungkan string
3. length() : Untuk menghitung total indeks pada sebuah array
4. toLowerCase() : Untuk mengubah string menjadi lower case 
5. toUpperCase() : Untuk mengubah string menjadi kapital
6. forEach() : Untuk memanggil setiap elemen di dalah array
7. join()   : Untuk menggabungkan semua elemen dalam array menjadi string
8. reverse() : Untuk membalik urutan indeks pada array
9. getDate() : Untuk mendapatkan tanggal sesuai waktu lokal
10. getTime() : Untuk mendapatkan waktu
